Welcome to the Storyteller blog

Posted on Oct 26, 2025

This repository now contains a minimal blog scaffold. It serves static HTML from the docs/ folder so GitHub Pages can publish it from the main branch using the /docs folder as the publishing source.

What you can do next

  • Edit docs/index.html to change the homepage layout.
  • Add future posts as HTML files in docs/posts/ and link them from the index.
  • Customize docs/assets/css/style.css to change visual styling.

If you prefer a generator (Jekyll/Hugo), we can convert this scaffold into one — let me know which you prefer.
